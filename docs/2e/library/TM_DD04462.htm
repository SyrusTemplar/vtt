Amulet of Metaspell Influence: This amulet does not appear to have any magical function (although it radiates magic if detected) until it is worn by someone using one of the dilation, far-reaching, or extension spells. When such a spell is cast, the amulet adds 50% to the functional effect of the spell. For example, if extension I is used to increase the duration of a 3rd-level spell by 50%, the wearer of this amulet can add one-half (50%) to that effect size, raising it to a 75% extension effect.
