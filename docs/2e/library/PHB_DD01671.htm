THAC0

is an acronym for -To Hit Armor Class 0.- This is the number a character, NPC, or monster needs to attack an Armor Class 0 target successfully. THAC0 depends on a character's group and level (see Table 53). The THAC0 number can be used to calculate the number needed to hit any Armor Class. THAC0 is refigured each time a character increases in level. Using THAC0 speeds the play of combat greatly.
