List of Kits

All of the bard kits are listed on the following pages. The first kit is that of the True Bard, as presented in the Player's Handbook. This kit establishes a base from which all the other kits are built. The other kits are arranged in alphabetical order. A section on multi-classed bards follows. It contains an additional four kits, but these are restricted to demihumans.
