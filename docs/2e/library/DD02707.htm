Does This Belong in My Campaign? While the rest of this book assumes that you-ll be playing with the full critical hit tables and rules, you-ll find that this chapter can be completely disregarded with no ill effects. More than anything else in this book, critical hits are optional; if the DM and players don-t want them, the AD&D game works fine without them.

However, there is one thing worth mentioning: if monsters have to suffer critical hits, so do player characters. Otherwise, game balance quickly goes out the window.
