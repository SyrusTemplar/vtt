See Sound

(clairsentient devotion)

MAC:
8
PSP Cost:
4/2
Range:
Special
Area of Effect:
Personal
Prerequisite:
None

This power enables a user to perceive sound waves visually by converting those waves into light impulses. Only a character who can see with normal vision can use this power. He can see sound even in darkness, as sound waves don’t require light. The user can be “blinded” by science, however.
