Saving Throw Penalties by Caster Level

Using Table 36 can be somewhat cumbersome to use in regular play because the DM must perform a calculation each time a spell is cast. Table 36 is intended primarily for magical dueling (see Chapter 5).

A quicker and easier method to adjust saving throws by caster level is to assign a penalty based on the caster?s level. Table 38 lists the penalties. If you use Table 38, do not use Table 36; use Table 60 from the Player?s Handbook instead.
