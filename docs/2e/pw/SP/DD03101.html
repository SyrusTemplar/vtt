Inherent Immunity/Disease: The character with this trait has a strong resistance to diseases of all types. When a saving throw is allowed against a possible infection, the character gains a +3 bonus. If there is an infection, the duration of the disease is reduced—by one day, week, etc.,—for each die rolled to determine this time period. This immunity does not apply to magical enchantments and curses, such as lycanthropy.
